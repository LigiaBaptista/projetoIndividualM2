O código Secreto

Você já imaginou criar um código secreto?

No campo abaixo você pode codificar ou descodificar o texto que quiser, utilizando a Base 64 ou a Cifra de César.

Para isso, basta selecionar qual código quer usar, se quer codificar ou descodificar, e digitar o texto que deseja que seja transformado.

Caso você opte pela Cifra de César, você precisa escolher a rotação.

Para descodificar você precisa saber o número da rotação do código que está criptografado.
